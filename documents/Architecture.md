# Project plan

## Frontend
tauri frameworked vanilla js desktop app.
keeps a list of some form of the software
updates with information from database periodically

Features:
searching, accepting/rejecting software (applications to update to client computers) clicking on software gives a popup with security information

## Backend

Using tauri framework, backend is built using Rust, managing and SQLite database and API requests to OpenAI API, and possibly Brave web search.

UI => keeps a list, that periodically loads from database

## techstack: 

TAURI tool for RUST backend and vanilla JS frontend
